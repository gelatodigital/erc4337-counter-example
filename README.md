# ERC-4337 Counter Example

## Quick Start

1. Install dependencies
   ```
   yarn install
   ```
2. Generate types
   ```
   yarn types
   ```
3. Edit ``.env``
   ```
   cp .env.example .env
   ```
## Different solutions
1. Zerodev 
   ```
   yarn zerodev
   ```
2. Pimlico Safe
   ```
   yarn pimlico
   ```
3. Gelato Safe
   ```
   yarn gelato
   ```
